# Java Training Repository

A comprehensive collection of Java modules covering core concepts, advanced features, and interview preparation topics.

## Overview

This repository contains educational Java code examples and demonstrations covering essential topics for Java developers. Each module focuses on specific concepts with practical examples and interview summaries.

## Modules

### Core Concepts
- **Java Basics** - Fundamental Java syntax and concepts
- **Interfaces vs Abstract Classes** - OOP abstraction patterns
- **Super vs This** - Reference keywords in inheritance
- **Final, Finally, Finalize** - Java termination concepts
- **Try-Catch-Finally** - Exception handling patterns
- **Throw vs Throws** - Exception propagation

### Collections & Data Structures
- **ArrayList vs Set** - List and Set implementations
- **Map vs Collections** - Map interface and utilities
- **Arrays vs Collections** - Primitive arrays vs Collections framework
- **Iterator vs ListIterator** - Collection traversal
- **Comparator vs Comparable** - Object comparison
- **HashMap Internal** - HashMap implementation details

### Object-Oriented Programming
- **HashCode and Equals** - Object identity and equality
- **Singleton Pattern** - Design pattern implementation

### Multithreading
- **Process vs Thread** - Concurrency concepts
- **Multithreading Basics** - Thread creation and management
- **Thread Lifecycle** - Thread states and transitions
- **Synchronized Block** - Synchronization mechanisms
- **Sync Read Write** - Read/write thread safety

### Modern Java Features
- **Stream API Basics** - Functional programming with streams
- **Java Version Features** - Evolution of Java language
- **Interface Types** - Modern interface features (default, static methods)

### Exception Handling
- **Exception Types** - Checked vs unchecked exceptions

## Running the Code

Each module can be compiled and run individually:

```bash
cd "Java Modules/<ModuleName>"
javac <ModuleName>.java
java <ModuleName>
```

Example:
```bash
cd "Java Modules/Multithreading_Basics"
javac Multithreading_Basics.java
java Multithreading_Basics
```
